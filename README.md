# 📊 Statistical Data Analysis & Distribution Modeling Project

## 📌 Overview

This project demonstrates the application of statistical concepts and probability distributions on a dataset using Python. It includes both theoretical explanations (Part A) and practical implementation (Part B).

---

## 📂 Dataset

The dataset (`spread_locator_dataset.csv`) contains transaction-related data used for statistical modeling and analysis.

---

## 🎯 Objectives

* Understand statistical distributions
* Perform data analysis using Python
* Fit data to different probability distributions
* Interpret statistical results for decision-making

---

## 🧠 Part A – Theoretical Foundation

Includes detailed explanations of:

* Statistical Distributions
* Q-Q Plot
* Discrete vs Continuous Distributions
* Bernoulli & Binomial Distribution
* Log-Normal & Power Law Distribution
* Box-Cox Transformation
* Poisson Distribution
* Z-score
* PDF & CDF

---

## 📊 Part B – Data Analysis & Testing

### ✔ Distribution Fitting

* Bernoulli & Binomial Distribution
* Poisson Distribution
* Log-Normal Distribution
* Power Law Distribution

### ✔ Statistical Methods

* Q-Q Plot (Normality Check)
* Box-Cox Transformation
* Z-score Calculation
* Probability Estimation

### ✔ Visualization

* PDF (Probability Density Function)
* CDF (Cumulative Distribution Function)
